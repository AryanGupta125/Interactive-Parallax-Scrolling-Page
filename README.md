# 🌌 Stellar Parallax

> An immersive, production-ready parallax scrolling website that takes users on a visual journey through cosmic landscapes, mountain vistas, ocean depths, and stellar galleries. Built with pure vanilla web technologies — no frameworks, no dependencies, just smooth 60fps magic.

Experience the web like never before. **Stellar Parallax** transforms ordinary scrolling into an engaging adventure where every section reveals new layers of depth, motion, and storytelling. From twinkling stars and drifting jellyfish to interactive tilt cards and animated counters, every detail is crafted to create a sense of wonder.

---

## ✨ Features

**Stellar Parallax** is more than just a website — it's a demonstration that **stunning, immersive web experiences don't require heavy frameworks or complex build tools**. By leveraging the power of modern HTML, CSS, and vanilla JavaScript, we've created a site that is fast, accessible, beautiful, and genuinely fun to scroll through.

The multi-layered parallax effects don't just look cool — they create emotional engagement, drawing users deeper into the content with each scroll. Combined with accessibility-first design and zero dependencies, Stellar Parallax proves that the **web platform itself is powerful enough** to deliver world-class experiences.

## 🛠️ Tech Stack

| Category | Technology |
|----------|-----------|
| **Markup** | HTML5 (Semantic) |
| **Styling** | CSS3 (Custom Properties, Grid, Flexbox, Transforms) |
| **Scripting** | Vanilla JavaScript (ES6+) |
| **Animation** | CSS Animations, `requestAnimationFrame`, CSS Transforms |
| **APIs Used** | Intersection Observer, MatchMedia, Performance API |
| **Graphics** | Inline SVG, CSS Gradients, Unsplash (gallery images) |
| **Icons** | Custom inline SVGs |
| **Fonts** | System font stack (Inter, -apple-system, Segoe UI) |
| **Dev Tools** | npx serve, live-server (optional) |

## 🚀 How to Run on Your Local System

### Option 1: 📂 Open Directly (Easiest)

Simply double-click `index.html` or open it in your browser:

✅ That's it! No installation needed.

---

### Option 2: 🌐 Local Development Server (Recommended)

A local server provides the best experience and avoids any CORS issues with images.

#### Using Node.js (recommended)

```bash
# Clone or download the repository
git clone https://github.com/your-username/stellar-parallax.git
cd stellar-parallax

# Install dev tools (optional)
npm install

# Start the server
npm start
```

Then visit 👉 **`http://localhost:8000`**
